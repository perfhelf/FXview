# 股指综合指数公式 (TAIXI 方法)

## 公式原理

将股指转换为"多货币标准化"视角，消除本币汇率波动干扰，得出股指的真实国际强度。

**通用公式结构**：
```
指数综合 = 指数/USD基准 + 指数/EUR基准 + 指数/GBP基准 + 指数/JPY基准 + 指数/AUD基准
```
